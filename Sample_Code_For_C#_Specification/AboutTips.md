## 类与对象的关系
对象也叫实例，是类经过“实例化”后的到内存中的实体  
有些类不能实例化  
依照类。我们可以创建对象，这就是实例化  
现实世界的对象在程序世界中成为实例  
使用new操作符创建类的实例  
引用变量与实例的关系  

```
Form myForm(); //引用变量
myForm = new Form(); //实例化
myForm.text="myForm";
myForm.ShowDialog();
```
```
Form myForm(); 
myForm1 = new Form(); 
myForm2 = new Form();
myForm2 = myForm1
myForm1.text="myForm1";
myForm2.text="myForm2";
myForm1.ShowDialog();  
```
这行代码 myForm2 = myForm1; 实际上是将一个对象引用赋值给另一个对象引用。在C#中，对象引用变量存储的是对象的内存地址，而不是对象本身的副本。因此，当你执行这行代码时，它会使myForm2引用和myForm1引用指向相同的对象，而不是创建一个新的对象。这意味着它们都指向同一个窗体对象。

## 变量、对象与内存  
**什么是变量**  
用来存储数据  
实际上变量表示存储位置，都有一个类型，决定什么值能存为变量  
有7个种类：静态、实例（成员、字段）、数组、值参数、引用参数、输出参数、局部变量  
狭义的变量指的是局部变量，方法体中声明的变量，*局部变量分配到栈中*  
**变量的声明**  
有效的的修饰符组合（可选）public static  
类型  
变量名  
初始化器（可选）指的是变量的设置操作比如赋值、运算等  
变量=以变量名所对应的内存地址为起点、以其数据类型所要求的存储空间为长度的一块内存区域  
**值类型的变量**  
值类型没有实例，所谓的实例和变量合二为一  
**引用类型的比哪里与实例**  
引用类型的变量存储的是引用类型的实例在堆内存上的内存地址  
通过这种方式引用类型变量构成了对引用类型实例的引用  
**变量的默认值**  
成员变量在声明后会有默认值0  
*值不可以改变的变量称为常量*  
const
**装箱和拆箱**  
损失性能
装箱 指的是将栈上的值类型的值封装obj类型的实例放在堆上  
拆箱 指的是将堆上obj类型的实例的值按照要求拆成目标类型存储到栈上去
