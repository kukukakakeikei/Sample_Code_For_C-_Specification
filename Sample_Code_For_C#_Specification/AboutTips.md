## 类与对象的关系
对象也叫实例，是类经过“实例化”后的到内存中的实体  
有些类不能实例化  
依照类。我们可以创建对象，这就是实例化  
现实世界的对象在程序世界中成为实例  
使用new操作符创建类的实例  
引用变量与实例的关系  

```
Form myForm(); //引用变量
myForm = new Form(); //实例化
myForm.text="myForm";
myForm.ShowDialog();
```
```
Form myForm(); 
myForm1 = new Form(); 
myForm2 = new Form();
myForm2 = myForm1
myForm1.text="myForm1";
myForm2.text="myForm2";
myForm1.ShowDialog();  
```
这行代码 myForm2 = myForm1; 实际上是将一个对象引用赋值给另一个对象引用。在C#中，对象引用变量存储的是对象的内存地址，而不是对象本身的副本。因此，当你执行这行代码时，它会使myForm2引用和myForm1引用指向相同的对象，而不是创建一个新的对象。这意味着它们都指向同一个窗体对象。
## 类的三大成员
属性（Property）  
存储数据，组合起来表示类或者对象当前的状态  
方法（Method）  
表示类或者对象能做什么，构成逻辑  
*数据加算法*  
事件（Event）  
类或者对象通知其它类或对象的机制，为C#所特有  
善用事件机制非常重要
